=============================================
INSTRUCCIONES DE INSTALACIÓN DE BASE DE DATOS
=============================================

El repositorio de GitHub para el Proyecto Coder se compone de 3 carpetas:
1. Documentación Técnica: Contiene información técnica adicional sobre el proyecto.
2. Documentación de Usuarios: Proporciona la documentación del proyecto destinada a los usuarios.
3. Script de Implementación: Contiene scripts necesarios para instalar la base de datos.

Para la creación exitosa de la Base de Datos "Proyecto Coder", se deben ejecutar los archivos que están en la carpeta "3 - Script de Implementación". Sigue los siguientes pasos en el orden propuesto:

PASO 1. Creación de Tablas:
- Contiene el script SQL para la creación de las tablas en la base de datos y la inserción de datos.
- Utiliza este script para iniciar la creación de la estructura de la base de datos.

PASO 2. Creación de Vistas:
- Utiliza el script `7 - B - Script de creación de vistas.sql` para agregar 7 vistas diferentes, las cuales son las siguientes:
     1. AlumnosActivos
     2. Consultas
     3. Material
     4. Movimiento
     5. Terminos
     6. VistaAlumnosUltimoPago
     7. VistaMaquinasActivas

PASO 3. Creación de Stored Procedures
- Ejecuta el script `8 - A - Script de creación de stored procedures.sql` para agregar 5 Procedimientos Almacenados, los cuales son los siguientes:
     1. BuscarRegistrosPesoPorDNI
     2. BuscarRegistrosPesoPorDNIConOrden
     3. GestionarInsercionMovimientos
     4. GestionarTransaccion
     5. InsertarRegistroPesoPorDNI

PASO 4. Creación de Funciones:
- Utiliza el script `8 - B - Script de creación de Funciones.sql` para incorporar funciones específicas del proyecto, las cuales son:
     1. CountMovimientos
     2. ObtenerUltimoPesoPorDNI

PASO 5. Creación de Triggers:
- Ejecuta `9 - Script de creación de triggers.sql` para implementar triggers y rastrear inserciones y cambios en ciertas tablas.

PASO 6. Implementación de Sentencias:
- Utiliza `10 - A - Implementación de Sentencias.sql` para crear dos usuarios: uno con permisos de solo lectura y el otro con permisos de Lectura, Inserción y Modificación de datos.

PASO 7. Sentencias del sublenguaje TCL:
- Ejecuta `10 - B - Sentencias del sublenguaje TCL.sql` para realizar modificaciones en ciertas tablas utilizando transacciones.

OBSERVACIONES: Se ha incluido un archivo llamado "Consultas a la base de datos.sql" en la carpeta de Documentación Técnica, con el cual se pueden realizar diversas consultas en las distintas tablas de la Base de datos.


¡Listo! Has completado la instalación de la base de datos para el proyecto. Asegúrate de revisar cualquier mensaje de error durante la ejecución de los scripts y verifica la integridad de la instalación.
